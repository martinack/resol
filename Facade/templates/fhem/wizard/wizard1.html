Jetzt Schnittstellen erkennen
Sehr gut, es wurde mindestens ein vbus gefunden.
Sie können die Einstellungen nun anpassen, z.B. wenn Sie anstelle der IP-Adresse einen DNS-Namen verwenden möchten, und anschließend auf "Speichern" klicken.
Leider wurde kein VBUS gefunden!
Dies kann verschiedene Ursachen haben:
  • Überprüfen Sie die Netzwerk Einstellungen
    Befindet sich das Gerät auf welchem FHEM installiert ist im selben Netz wie der VBUS?
  • Überprüfen Sie die VBUS Verkabelung
    Ein häufig auftretendes Problem ist, das vertauschen von Kabelenden.
  • Ist ihre VBUS-Firmware komptibel?
    Aktuell werden nur die Versionen {@TODO: add supported versions} unterstützt

Eine weitere Ursache könnte ein unbekannter Fehler der automatischen Geräte Erkennung sein.
Deshalb haben Sie auch die Möglichkeit diesen Schritt manuell durchzuführen.
Tragen Sie dafür in den unten stehenden Feldern die entsprechenden Daten ein, und klicken Sie anschließend auf "Speichern".
Wenn der nächste Schritt problemlos verläuft, lag ein Fehler in der automatischen Geräte Erkennung vor.
Sie können die hier vorgenommenen Einstellungen später jeder Zeit wieder ändern.

Wenn Sie ihre Einstellungen vorgenommen und gespeichert haben, klicken Sie bitte auf "Weiter".
